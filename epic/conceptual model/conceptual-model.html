Conceptual Model

Profile

  • profileId (Primary Key)
  • profileActivationToken
  • profileAddress
  • profileAvatarUrl
  • profileEmail
  • profileHash
  • profileType
  • profileUsername

Delivery

  • deliveryId (Primary Key)
  • deliveryDate
  • deliveryStatus (promised and shipped)
  • deliveryTrackNumber
  • deliveryService

Request

  • requestId (Primary Key)
  • requestContent
  • requestDate
  • requestProfileId (Foreign Key)

Item

  • itemId (Primary Key)
  • itemDate
  • itemProfileId (Foreign Key)
  • itemRequestId (Foreign Key)
  • One profile can make many request
  • One request can have many items
  • One profile can accept many items
