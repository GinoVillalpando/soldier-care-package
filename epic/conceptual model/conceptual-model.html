Conceptual Model

Profile

  • profileId (Primary Key)
  • profileActivationToken
  • profileAddress
  • profileAvatarUrl
  • profileBio
  • profileEmail
  • profileHash
  • profileType
  • profileUsername

Delivery

  • packageId (Primary Key)
  • packageDate
  • packageProfileId (Foreign Key)
  • packageService
  • packageStatus (promised and shipped)
  • packageTrackNumber

Request

  • requestId (Primary Key)
  • requestContent
  • requestDate
  • requestProfileId (Foreign Key)

Item

  • itemId (Primary Key)
  • itemDate
  • itemPackageId (Foreign Key)
  • itemRequestId (Foreign Key)
  • itemUrl
  • One profile can make many request
  • One profile can have many packages
  • One request can have many items
  • One package can have many items
